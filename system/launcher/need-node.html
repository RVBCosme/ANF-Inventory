One quick thing first

ANF Inventory needs a free helper called Node.js installed on this computer. This is a one-time step.

  1. Click the button below to open the Node.js website.
  2. Download the LTS version and run the installer with the default options.
  3. When it finishes, start ANF Inventory again.
